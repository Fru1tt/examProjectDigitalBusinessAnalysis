# Notebooks

Use notebooks for exploration and prototyping.

Naming convention:
- `01_data_understanding.ipynb`
- `02_eda.ipynb`
- `03_model_or_analysis.ipynb`
- `04_visual_story.ipynb`

Guidelines:
- Keep one clear objective per notebook.
- Move stable logic into `scripts/`.
- Export final visuals to `outputs/figures/`.
